# drowsiness-detection

The goal is to develop a real-time drowsiness detection system that monitors individuals in a video feed. The system will employ a two-step process: initially, it will detect faces in the video. Once a face is detected, it will classify the individual's state as drowsy, low vigilant, or awake using a pretrained model. This model, trained on a dataset featuring alert, low vigilant, and drowsy individuals, will provide a probability score for each class. Based on these scores, the system will apply a threshold to determine the individual's state and provide localization of the class through bounding boxes. This approach can significantly enhance driver and workplace safety. However, the system's accuracy will hinge on the effectiveness of the face detection algorithm, the quality and diversity of the training data, and the appropriateness of the chosen threshold.
